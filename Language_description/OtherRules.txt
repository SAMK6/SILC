Features wishlist
- named SILC "Sam and Isaac Like C"
- compiled language
- SILC files have .silc extensions
- types
  - integer (8 bytes, signed)
  - character (1 byte, unsigned)
- keywords (strictly reserved words)
  - in "integer"
  - ch "character"
  - pr "print"
- operators
  - +
  - -
  - ^
  - *
  - ~
  - =
- misc
  - lines end with ';' always
  - white spaces is stripped during compilation
  - declaration is always explicit so variables must be declared on a line before they're assigned
- variables
  - [a-z|A-Z|0-9|_]+ no more than 16 characters
